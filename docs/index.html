功能特性

📍

极简侧边导航

右侧窄条标记所有提问，鼠标移入向左展开完整面板，点击快速定位到对应问题。

🌓

自动暗色模式

根据页面主题自动切换深浅配色，无缝融入各平台的界面风格。

⚡

虚拟滚动适配

针对 Kimi、豆包等使用虚拟滚动的平台做了专门优化，点击跳转稳定可靠。

🔒

Shadow DOM 兼容

支持豆包等使用 Shadow DOM 的平台，穿透 Shadow boundary 查找消息元素。

🎯

即时定位

无动画延迟，点击后页面瞬间滚动到目标位置，浏览长对话更高效。

🔌

一键开关

通过扩展弹窗随时开启或关闭导航功能，不影响正常的对话体验。

安装方式

  1. 从 GitHub Releases 下载最新版本的 chatnest-v*.zip
  2. 解压 zip 文件，得到 dist 文件夹
  3. 打开 Chrome 扩展管理页 chrome://extensions/
  4. 开启右上角「开发者模式」
  5. 点击「加载已解压的扩展程序」，选择解压后的 dist 文件夹
